居中方法和原理总结

2021-03-18

引言：居中(水平 垂直)需求大，这里总结一下居中的几种方式和对应的原理

  1. translate实现水平和垂直居中
    translate(x,y) 根据自身向x ， y 移动 若为百分比 则数值为自身的宽和高
position: absolute
left:50%;
top:50%,
translate(-50%,-50%)

left:50%; top:50%, 实现要居中的内容左上角到达父元素中心
translate(-50%,-50%) 根据自身宽高移动一半 实现完美居中
tip：外盒子需要被内盒子定位到 所以需要relative定位
2. margin：0 auto 水平居中
适用：定宽的block盒子
block 块状独占一行 如果不定宽则默认100% 也没有auto默认值了
auto值是浏览器自动判断 在此处会平分剩余空间
从而达到水平居中效果
tip： 如果是行内元素可以改成block进行使用margin进行水平居中
3. text-align 水平居中（对内容进行对齐）
text-align CSS属性定义行内内容（例如文字）如何相对它的块父元素对齐。text-align 并不控制块元素自己的对齐，只控制它的行内内容的对齐。 ——MDN
其实经过实践，text-align的特点有

  • 具有继承性
  • 对设置其属性的行内级子元素具有居中效果
    因为具有继承性，无论是块状还是行内的内联元素都能进行居中 所以所有的行内子元素都能进行居中
  1. display: flex; justify-content: center; 水平居中
    flex布局的一个内置排列方法
    （稍微看了下flex布局 另起一篇进行学习）
    justify-content: center 伸缩元素向每行中点排列。每行第一个元素到行首的距离将与每行最后一个元素到行尾的距离相同。
    设置为flex属性的元素为 flex容器 而其所有的子元素称为 flex项目 flexitem
    justify-content 还有很多排列方法 不一一探讨
  2. line-height 垂直居中

标准定义：两行文字基线之间的距离。

有以下要点

  • 在 CSS 中，每一行文字也可以看成一个“行盒子”，而每一个行盒子都有4 条线：顶线、 中线、基线和底线。 内容高度为文本高度 上下距离（line-height - 文本高度） /2
  • line-height=上距离+内容高度+下距离
  • 上下距离是一定会一样的 上距离==下距离
  • 某些时候（单文本）可以不用设置height 因为line-height会撑开盒子 设置height与line-height相等多此一举
  • 可以被继承
    了解完这些 也可以理解为什么line-height可以进行垂直居中了
  1. varticle-align 垂直居中
    父元素设置以下两个属性进行table布局
    display: table-cell;
    vertical-align:middle;
  2. align-items: center; 垂直居中
    给待处理的块状元素的父元素添加属性 display: flex; align-items: center;
  3. top: 0; right: 0; bottom: 0; left: 0; margin: auto; 水平和垂直居中
    对父元素设置position：relative 子元素设置position： absolute ->此时子元素已经变成块级元素
    再对子元素设置top: 0; right: 0; bottom: 0; left: 0; margin: auto; （子元素进行定高宽）
  4. grid 网格布局内置水平 垂直居中方法
    align-items: center; 内容垂直居中在单元格里
    justify-content: center; 内容整体水平居中在grid容器

总结：里面还有一些方法可以自由转换 如flaot隐性转化成块状等等 在父相子绝中 margin代替 transform... 目前还缺少实践 无法总结出每种方式的优缺点和比较适用的场景 之后再回来修改~
